# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Read the Room, please report it
responsibly. **Do not open a public GitHub issue.**

Email **dev@readtheroom.site** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if you have one)

You should receive an acknowledgment within 48 hours. Please allow reasonable
time for a fix before any public disclosure.

## Scope

The following are in scope:

- The Read the Room Flutter app (this repository)
- Supabase Edge Functions and database configuration
- Authentication and passkey implementation
- Data privacy and account deletion

The following are out of scope:

- Third-party services (Supabase, Firebase, PostHog) — report to those providers
- Denial-of-service attacks
- Social engineering

## Supported Versions

Only the latest release is supported with security updates.
